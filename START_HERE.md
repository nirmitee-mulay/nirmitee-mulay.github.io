## ⚡ Fastest Path to Deployment (5 Steps)

### 1. Download All Files
Download this entire folder to your computer.

### 2. Add Your Images
Create these directories and add your files:
```
images/
  └── profile.jpg          (your photo, 800x800px)
  └── art/
      ├── piece1.jpg
      ├── piece2.jpg
      └── ... (your artwork)
files/
  └── Nirmitee_Mulay_CV.pdf  (your CV as PDF)
```

### 3. Update Your Links
Search for "#" in all HTML files and replace with your actual:
- ORCID ID
- Google Scholar profile URL  
- Bluesky handle
- Art portfolio link

### 4. Deploy to GitHub
```bash
# Clone your repo
git clone git@github.com:nirmitee-mulay/nirmitee-mulay.github.io.git
cd nirmitee-mulay.github.io

# Copy all files here
# (drag and drop or use cp command)

# Deploy
git add .
git commit -m "Initial website"
git push origin main
```

### 5. Enable GitHub Pages
1. Go to https://github.com/nirmitee-mulay/nirmitee-mulay.github.io/settings
2. Click "Pages" in left sidebar
3. Set Source to "Deploy from branch: main / (root)"
4. Click Save

**Your site will be live at https://nirmitee-mulay.github.io in 2-5 minutes!**
