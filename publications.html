PREPRINT

Unmasking Unconscious Working Memory using a Bias-Free Face Ensemble Paradigm

Mulay, N. N., de Castro Urquiola, I., Abdennour, N., Elosegi, P., & Soto, D.

PsyArXiv (December 2025)

We developed a bias-free paradigm to test whether visual ensemble representations can be maintained in working memory without conscious awareness. Using 2IFC tasks and face ensembles, we found no evidence for unconscious working memory during maintainance in presence of distracto information. This challenges previous claims and highlights the importance of methodological rigor in consciousness research.

PUBLISHED

Lifting the veil: Altered visual perception in derealisation

Kusztor, A., Mulay, N. N., Yamada, M., Hohwy, J., & Tsuchiya, N.

[Journal details to be added] (2023)

This study investigated whether the subjective visual alterations reported in derealisation (DR)—a dissociative condition where the world appears "unreal" or distorted—correspond to measurable changes in basic visual perception. Using psychophysical tasks, we assessed visual processing in individuals with DR traits.
