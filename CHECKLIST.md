# Website Customization Checklist

Use this checklist to ensure you've customized all necessary parts of the website before deploying.

## 📋 Before Deployment

### Essential Updates

- [ ] **Profile Photo**
  - [ ] Add your photo as `images/profile.jpg` (800x800px recommended)
  - [ ] Verify it displays correctly on the homepage

- [ ] **CV PDF**
  - [ ] Export your LaTeX CV as PDF
  - [ ] Save as `files/Nirmitee_Mulay_CV.pdf`
  - [ ] Test the download link on cv.html

- [ ] **Contact Links** (Update in ALL pages)
  - [ ] Add ORCID ID (search for "#" placeholder)
  - [ ] Add Google Scholar profile link
  - [ ] Add Bluesky handle
  - [ ] Add art portfolio link (if you have one)
  - [ ] Verify all email addresses are correct

### Content to Add

- [ ] **Publications Page**
  - [ ] Add talk title for ASSC 2025 presentation
  - [ ] Add thesis talk title from IISER Kolkata
  - [ ] Add poster title from Qualia Structure meeting
  - [ ] Add DOI/journal for derealization paper
  - [ ] Add poster link if available

- [ ] **Research Page**
  - [ ] Review project descriptions for accuracy
  - [ ] Add specific computational modeling interests
  - [ ] Update "Looking Ahead" section with target labs/PIs if desired

- [ ] **About Me (about_me.md)**
  - [ ] Fill in bracketed sections with specific details
  - [ ] Add ORCID, Google Scholar, other profile links
  - [ ] Update research interests list
  - [ ] Complete "Looking Ahead" section

### Art & Images

### Optional Enhancements

- [ ] **Color Scheme**

  - [ ] Test color contrast for accessibility

## 🚀 Deployment Steps

- [ ] Clone your GitHub repository locally
- [ ] Copy all website files to the repository
- [ ] Create directory structure (images/, files/, images/art/)
- [ ] Add your images and CV PDF
- [ ] Review all pages in a browser locally
- [ ] Commit and push to GitHub
- [ ] Enable GitHub Pages in repository settings
- [ ] Wait 2-5 minutes for site to go live
- [ ] Test site at https://nirmitee-mulay.github.io
- [ ] Check all pages on mobile device

## ✅ Post-Deployment Verification

- [ ] Mobile menu works
